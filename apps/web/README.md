# Ticketing System Web App

Frontend for the IT Ticketing System built with Next.js, React, and Tailwind CSS.

## Features

- Authentication protected routes
- Department dashboard
- Ticket management UI
- Assign, escalate, resolve, close tickets
- Activity log timeline
- Role-based actions

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- SweetAlert2

## Setup

### Install dependencies
npm install

### Environment variables
Create `.env.local`:

NEXT_PUBLIC_API_URL=http://localhost:5000

### Run development server
npm run dev

App runs at:
http://localhost:3000
